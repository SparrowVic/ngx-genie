Angular compatibility

GenieOS tracks the Angular release train closely. The current line targets Angular 21, with maintained support branches reaching back to Angular 17.

Angular 21 · recommended Angular 20 · legacy line Angular 17–19 · supported

Dev-only by design

To reconstruct the injector graph, GenieOS reaches into private Angular DI internals. That access is centralised, re-verified on every upgrade by a compatibility test suite, and tree-shaken out of production builds. Install it as a devDependency and never ship it to end users.
